## ğŸ“Š Understanding the Visualizations

### SIT Role Map
- **Bubble size** = Number of actors in that role
- **Arrows** = Flow direction (products, money, information)
- **Use:** Identify missing links in value chain

### SAT Power-Interest Matrix
- **Top-right quadrant** = Manage closely (key players)
- **Top-left quadrant** = Keep satisfied (high power)
- **Bottom-right quadrant** = Keep informed (high interest)
- **Bottom-left quadrant** = Monitor (minimal effort)

### MAT Complaint Analysis
- **Pie chart** = Distribution by category
- **Bar chart** = Distribution by severity
- **Use:** Prioritize improvement areas
